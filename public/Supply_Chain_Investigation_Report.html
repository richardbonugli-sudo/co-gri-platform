Detailed Findings

Question 1: What datasets are feeding the SupplyChain channel?

Answer: Primary Source is SUPPLY_CHAIN_INTENSITY matrix with 4 countries (US, China, Germany, Japan), ~20 country pairs. It is NOT trade-derived.

Question 2: Is the fallback logic generating proxy values or defaulting to zero?

Answer: The fallback logic is defaulting to zero when direct data is missing. This is a false zero - it means no data not no relationship.

Question 3: Is the supply-chain logic truly symmetric?

Answer: The logic IS symmetric in implementation, but the data coverage gap creates asymmetric results.
